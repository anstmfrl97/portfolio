My Part

배너 제작

카테고리(테마) :
자연, 문화예술, 음식 등 여러 주제 중,
흥의 민족답게 음악페스티벌을 주제로 제작하였습니다.

카테고리(시즌) :
각 시즌별 자연축제를 주제로 배너를 제작하였습니다.

메인 배너 :
축제를 즐기는 여러 사람들의 모습을 담아 신나고 역동적인 분위기를 주었습니다.

로고 제작

불꽃축제를 컨셉으로 '넘쳐흐르는 수많은 축제들, 그 안의 우리' 라는 뜻을 담아서 로고를 제작하였습니다.

장식선 제작

카테고리, 검색, Footer 장식선 :
음악 페스티벌과 네온사인의 화려함을 담았습니다.

Preview

Preview

생동감있는 느낌을 주기 위해 메인 배너를 gif로 만들어 배치하였고,

메인 페이지의 메뉴들을 Tap기능으로 구성하여 원하는 주제의 포스터를 볼 수 있게 의도하였습니다.

메인에 검색창을 배치하여 쉽게 원하는 축제를 찾고
실시간 검색순위로 인기축제를 한눈에 볼 수 있도록 디자인하였습니다.

웹페이지 내에 커뮤니티를 형성하여 유저들이 후기나 양도, 꿀팁 등을 공유하기 편리하도록 하였습니다.
